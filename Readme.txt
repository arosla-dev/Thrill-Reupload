Title: Thrill

URL: https://www.moddb.com/mods/thrill

Installation:

Put "thrill" folder to the root of your Steam version of Half-Life. Restart Steam client for the mod to appear in library.

Known issues:

- HUD sprites won't display properly on software renderer due to the usage of TriAPI
